== Installation ==

It's easy to get started

1. Upload `ht-social-widget` to the `/wp-content/plugins/` directory or goto Plugins>Add New and search for Heroic Social Widget.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Select Settings>Heroic Social Widget to set up the plugin with active icons and urls.
4. Click icons to activate them and drag and drop to change the order they appear.
5. Finally, in Appearance>Widgets you can select the Heroic Media Widget and add it to any widget area to display the active social icons.
